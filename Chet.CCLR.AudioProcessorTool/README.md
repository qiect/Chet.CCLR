# WPF音频标签工具使用说明

## 项目概述

WPF音频标签工具是一个用于音频时间轴标注的桌面应用程序,支持添加、编辑、删除标签,并导出标签文件。

## 功能特性

### 1. 音频播放控制
- **播放/暂停**：空格键或点击按钮控制音频播放
- **停止**：停止当前播放
- **实时进度显示**：显示当前播放时间和总时长
- **进度条拖拽**：通过Slider控制播放位置

### 2. 标签管理
- **添加标签**：按空格键或点击"添加标签"按钮,自动记录当前时间点
- **编辑标签**：选择标签后可编辑开始/结束时间和内容
- **删除标签**：删除不需要的标签
- **标签列表**：显示所有标签的序号、时间、时长和内容

### 3. 数据导出
- **导出标签文件**：生成labels.txt格式文件
- **导出格式**：开始时间\t结束时间\t序号

## 使用流程

### 第一步：准备文件
1. 准备音频文件(支持MP3、WAV、M4A等格式)

### 第二步：加载文件
1. 运行AudioTagger.exe
2. 点击"选择音频文件"选择音频文件
3. 点击"加载"按钮

### 第三步：打标签
1. 点击"播放/暂停"开始播放音频
2. 当听到需要标注的位置时,按空格键或点击"添加标签"
3. 工具会自动记录当前时间
4. 重复步骤2-3直到所有需要的标签都添加完成

### 第四步：编辑和调整
1. 在标签列表中选择某个标签
2. 点击"编辑标签"按钮
3. 在弹出的对话框中修改开始时间、结束时间或内容
4. 点击"确定"保存修改
5. 可以删除错误的标签后重新添加

### 第五步：试听时间段
1. 在标签列表中选择某个标签
2. 标签会自动跳转到开始时间位置
3. 可以点击播放按钮试听该时间段

### 第六步：导出标签
1. 点击"导出标签"按钮
2. 选择保存位置和文件名(默认为labels.txt)
3. 工具会生成标签文件

## 技术架构

### 核心组件
- **NAudio**：用于音频播放和处理
- **WPF**：用于UI界面
- **.NET 10**：运行时环境

### 数据结构
```csharp
public class TagItem
{
    public int Index { get; set; }      // 序号
    public double StartTime { get; set; }  // 开始时间(秒)
    public double EndTime { get; set; }    // 结束时间(秒)
    public double Duration { get; }        // 时长
    public string Content { get; set; }    // 标签内容
}
```

### 导出格式
```
0.000000	5.230000	001
5.230000	10.450000	002
10.450000	15.670000	003
```

格式说明：
- 第1列：开始时间(秒,6位小数)
- 第2列：结束时间(秒,6位小数)
- 第3列：序号(3位数字)

## 依赖项

## 注意事项

1. **音频格式支持**：推荐使用MP3或WAV格式,确保兼容性
2. **时间精度**：标签时间精度为毫秒级(0.001秒)
3. **播放器限制**：当前版本不支持跳转到指定时间点,只能通过拖拽进度条粗略控制

## 常见问题

### Q1: 如何修正错误的标签？
A: 在标签列表中选择错误的标签,点击"编辑标签"修改时间,或点击"删除标签"后重新添加。

### Q2: 导出的标签文件格式是什么？
A: 导出的标签文件是制表符分隔的文本文件,格式为：开始时间\t结束时间\t序号

### Q3: 支持哪些音频格式？
A: 支持MP3、WAV、M4A、AAC、FLAC等常见音频格式

### Q4: 如何试听某个时间段的音频？
A: 在标签列表中选中该标签,然后点击播放按钮即可试听该标签的时间段。
