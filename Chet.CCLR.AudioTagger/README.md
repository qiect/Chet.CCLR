# WPF音频标签工具使用说明

## 项目概述

WPF音频标签工具是一个专门用于音频句子时间轴标注的桌面应用程序，用于配合AudioProcessor进行音频拆分处理。

## 功能特性

### 1. 音频播放控制
- **播放/暂停**：空格键或点击按钮控制音频播放
- **停止**：停止当前播放
- **实时进度显示**：显示当前播放时间和总时长
- **进度条拖拽**：通过Slider控制播放位置

### 2. 标签管理
- **添加标签**：按空格键或点击"添加标签"按钮，自动记录当前时间点
- **编辑标签**：选择标签后可编辑句子内容
- **删除标签**：删除不需要的标签
- **标签列表**：显示所有标签的序号、时间、时长和内容

### 3. 句子管理
- **句子文件加载**：支持从文本文件加载句子列表
- **句子内容编辑**：实时编辑句子内容
- **自动更新**：标签与句子内容自动关联

### 4. 数据导出
- **导出标签文件**：生成AudioProcessor所需的labels.txt格式
- **保存句子文件**：保存编辑后的句子内容

## 使用流程

### 第一步：准备文件
1. 确保已运行AudioProcessor进行文本分句
2. 获取分句结果文件（sentences.txt）
3. 准备音频文件（支持MP3、WAV、M4A等格式）

### 第二步：加载文件
1. 运行AudioTagger.exe
2. 点击"选择音频文件"选择音频文件
3. 点击"选择句子文件"选择sentences.txt
4. 点击"加载"按钮

### 第三步：打标签
1. 点击"播放/暂停"开始播放音频
2. 当听到句子结束时，按空格键或点击"添加标签"
3. 工具会自动记录当前时间并移动到下一个句子
4. 重复步骤2-3直到所有句子都打上标签

### 第四步：编辑和调整
1. 在标签列表中选择某个标签
2. 在右侧编辑框中修改句子内容
3. 点击"更新句子内容"保存修改
4. 可以删除错误的标签后重新添加

### 第五步：导出标签
1. 点击"导出标签"按钮
2. 选择保存位置和文件名（默认为labels.txt）
3. 工具会生成符合AudioProcessor格式的标签文件

### 第六步：音频切割
1. 将导出的labels.txt放入AudioProcessor的output目录
2. 运行AudioProcessor并选择"执行音频切割"
3. 工具会自动按标签时间轴切割音频

## 技术架构

### 核心组件
- **NAudio**：用于音频播放和处理
- **WPF**：用于UI界面
- **.NET 10**：运行时环境

### 数据结构
```csharp
public class TagItem
{
    public int Index { get; set; }      // 序号
    public double StartTime { get; set; }  // 开始时间（秒）
    public double EndTime { get; set; }    // 结束时间（秒）
    public double Duration { get; }        // 时长
    public string Content { get; set; }    // 句子内容
}
```

### 导出格式
```
0.000000	5.230000	001
5.230000	10.450000	002
10.450000	15.670000	003
```

格式说明：
- 第1列：开始时间（秒，6位小数）
- 第2列：结束时间（秒，6位小数）
- 第3列：序号（3位数字）

## 依赖项

- **NAudio 2.2.1**：音频处理库
- **.NET 10.0 Windows**：运行时环境

## 注意事项

1. **音频格式支持**：推荐使用MP3或WAV格式，确保兼容性
2. **时间精度**：标签时间精度为毫秒级（0.001秒）
3. **文件编码**：句子文件使用UTF-8编码
4. **播放器限制**：当前版本不支持跳转到指定时间点，只能通过拖拽进度条粗略控制

## 常见问题

### Q1: 为什么添加标签后没有自动移动到下一个句子？
A: 这是正常行为，您可以手动点击下一个句子进行编辑。

### Q2: 如何修正错误的标签？
A: 在标签列表中选择错误的标签，点击"删除标签"，然后在正确的位置重新添加。

### Q3: 导出的标签文件格式是什么？
A: 导出的标签文件是制表符分隔的文本文件，格式为：开始时间\t结束时间\t序号

### Q4: 支持哪些音频格式？
A: 支持MP3、WAV、M4A、AAC、FLAC等常见音频格式
