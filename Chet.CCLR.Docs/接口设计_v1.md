# 一、认证模块

小程序登录后必须获取用户身份。

---

## 1️⃣ 微信登录

### POST `/api/auth/wx-login`

### 请求参数：

```json
{
  "code": "wx.login返回的code"
}
```

### 返回：

```json
{
  "token": "jwt-token",
  "user": {
    "id": "u001",
    "nickname": "",
    "avatarUrl": ""
  }
}
```

说明：

- 服务端换取 openid
- 若不存在则创建 user
- 返回 JWT

# 二、书目模块

---

## 2️⃣ 获取书目列表

### GET `/api/books`

返回：

```json
[
  {
    "id": "book01",
    "title": "论语",
    "category": "儒家",
    "description": "",
    "coverUrl": ""
  }
]
```

---

## 3️⃣ 获取章节列表

### GET `/api/books/{bookId}/chapters`

---

## 4️⃣ 获取章节句子列表（核心接口）

### GET `/api/chapters/{chapterId}/sentences`

返回：

```json
[
  {
    "id": "s001",
    "content": "学而时习之，",
    "pinyin": "xué ér shí xí zhī",
    "note": "学习并时常温习",
    "audioUrl": "https://xxx/001.mp3",
    "durationSec": 4,
    "orderIndex": 1
  }
]
```

⚠ 这个接口是听读页核心数据源。

---

# 三、听读进度模块

---

## 5️⃣ 获取当前书籍听读进度

### GET `/api/listen/progress?bookId=book01`

返回：

```json
{
  "bookId": "book01",
  "sentenceId": "s012",
  "progressSec": 3
}
```

若无记录返回 null。

---

## 6️⃣ 保存听读进度

### POST `/api/listen/progress`

```json
{
  "bookId": "book01",
  "sentenceId": "s012",
  "progressSec": 3
}
```

说明：

- 页面退出时调用
- 暂停时调用
- 每30秒自动保存一次（推荐）

---

# 四、听读统计模块

---

## 7️⃣ 上报听读时长

建议采用“会话式统计”。

### POST `/api/listen/record`

```json
{
  "bookId": "book01",
  "durationSec": 180
}
```

逻辑：

- 后端按日期累计
- 若当天 ≥ 180 秒记为有效一天

---

## 8️⃣ 获取统计信息

### GET `/api/user/stats`

返回：

```json
{
  "todayDurationSec": 420,
  "totalDurationSec": 15200,
  "continuousDays": 7
}
```

连续天数：

- 服务端计算
- 不存在数据库字段

# 八、接口调用流程图（听读页）

进入听读页：

1. 获取章节句子列表
2. 获取当前进度
3. 定位句子
4. 开始播放

播放中：

- 每播放完一句 → 本地 index++
- 每30秒 → 调用保存进度
- 退出时 → 保存进度 + 上报时长
