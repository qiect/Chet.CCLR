# 三、音频切片方案

当你有时间轴后，用 FFmpeg 自动切音频。

---

## FFmpeg 切片命令

```bash
ffmpeg -i book.mp3 -ss 00:01:20 -to 00:01:28 -c copy sentence_001.mp3
```

.NET 调用：

```csharp
public void CutAudio(string input, string output, double start, double end)
{
    var args = $"-i \"{input}\" -ss {start} -to {end} -c copy \"{output}\"";

    var process = new Process();
    process.StartInfo.FileName = "ffmpeg";
    process.StartInfo.Arguments = args;
    process.Start();
    process.WaitForExit();
}
```

# 四、数据库导入脚本生成器设计

## 生成 SQL 脚本

```csharp
public string GenerateSentenceInsertSql(List<Sentence> sentences)
{
    var sb = new StringBuilder();

    foreach (var s in sentences)
    {
        sb.AppendLine($@"
INSERT INTO sentence
(classic_id, chapter_id, sequence_no, content, pinyin, audio_url, start_ms, end_ms)
VALUES
({s.ClassicId}, {s.ChapterId}, {s.Sequence}, N'{s.Content}', N'{s.Pinyin}', '{s.AudioUrl}', {s.Start}, {s.End});
");
    }

    return sb.ToString();
}
```

输出：

```
import_20260211.sql
```

# 八、推荐长期最佳实践

⚠️ 强烈建议：

> 永远保留整本音频
> 切片音频只是优化

播放时：

- 如果 sentence 有独立音频 → 用切片
- 没有 → 用整本音频 + seek 播放

这样未来扩展成本最低。
