# 快速使用指南

## 3分钟快速上手

### 第一步：准备（1分钟）
1. 确保已运行AudioProcessor进行文本分句
   ```bash
   启动AudioProcessor.bat
   ```
2. 分句结果已保存在 `output/sentences.txt`

### 第二步：打标签（1分钟）
1. 运行AudioTagger
   ```bash
   启动AudioTagger.bat
   ```
2. 选择音频文件：`道德经_全文朗读_王弼本.m4a`
3. 选择句子文件：`output/sentences.txt`
4. 点击"加载"

### 第三步：开始打标签（1分钟）
1. 点击"播放/暂停"开始播放
2. 听到句子结束时，按空格键添加标签
3. 重复直到所有句子都打上标签
4. 点击"导出标签"保存为 `labels.txt`

### 第四步：切割音频（1分钟）
1. 将 `labels.txt` 放入 `output` 目录
2. 运行AudioProcessor
3. 选择"执行音频切割"
4. 完成！

## 核心功能

| 功能 | 快捷键 | 说明 |
|------|--------|------|
| 播放/暂停 | 空格 | 播放或暂停音频 |
| 添加标签 | 空格（播放时） | 记录当前时间点 |
| 停止 | - | 停止播放 |
| 导出标签 | - | 生成labels.txt |

## 注意事项

1. **音频文件**：支持MP3、WAV、M4A等格式
2. **句子文件**：UTF-8编码的文本文件
3. **时间精度**：毫秒级（0.001秒）
4. **标签数量**：应与句子数量一致

## 常见问题

### Q: 按空格键没有反应？
A: 确保音频正在播放状态下按空格键。

### Q: 标签时间不准确？
A: 可以在添加标签后进行微调，或删除后重新添加。

### Q: 导出的标签文件格式？
A: 格式为：开始时间\t结束时间\t序号（制表符分隔）
