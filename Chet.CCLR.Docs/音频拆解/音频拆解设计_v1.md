# 三、第一步：文本自动分句（.NET 实现）

你可以写一个分句服务：

```csharp
public static List<string> SplitSentences(string text)
{
    var pattern = @"(?<=[。！？；])";
    var parts = Regex.Split(text, pattern)
                     .Select(x => x.Trim())
                     .Where(x => !string.IsNullOrWhiteSpace(x))
                     .ToList();
    return parts;
}
```

如果是启蒙类想按逗号拆：

```csharp
var pattern = @"(?<=[，。！？；])";
```

然后人工检查一遍。

⚠ 这一步必须人工确认。

# 方案 A（推荐）：Audacity 打标签 + 导出标签时间文件

操作：

1. 导入整本音频
2. 每句结束 Ctrl+B 打标签
3. 文件 → 导出 → 导出标签
4. 得到一个 .txt 文件

格式类似：

```

4.230000    8.910000    002

```

这就是你所有时间段。

这个文件可以交给 .NET 处理。

# 五、第三步：使用 FFmpeg 批量切割（推荐做法）

不要用 C# 自己处理音频流。

直接调用 FFmpeg。

## 为什么？

- 快
- 稳
- 跨平台
- 专业

---

## .NET 调用 FFmpeg 示例

```csharp
public static void CutAudio(string sourceFile, string outputFile, double start, double duration)
{
    var args = $"-i \"{sourceFile}\" -ss {start} -t {duration} -acodec copy \"{outputFile}\" -y";

    var process = new Process
    {
        StartInfo = new ProcessStartInfo
        {
            FileName = "ffmpeg",
            Arguments = args,
            RedirectStandardOutput = true,
            RedirectStandardError = true,
            UseShellExecute = false,
            CreateNoWindow = true
        }
    };

    process.Start();
    process.WaitForExit();
}
```

## 批量处理逻辑

读取标签文件：

```csharp
var lines = File.ReadAllLines("labels.txt");

foreach (var line in lines)
{
    var parts = line.Split('\t');

    double start = double.Parse(parts[0]);
    double end = double.Parse(parts[1]);
    string name = parts[2];

    double duration = end - start;

    CutAudio("book.mp3", $"output/{name}.mp3", start, duration);
}
```

这样：

👉 1小时音频 300句

几分钟全部切完。

# 六、一本书一个音频 vs 一章一个音频

## 情况 1：一本书一个音频

没问题：

- 全书统一编号
- sentence 按全书顺序

例如：

```
book01_001.mp3
book01_002.mp3
```

数据库 sentence 直接对应。

---

## 情况 2：一章一个音频

建议：

- 每章单独打标签
- 每章单独切割
- 编号从 001 开始

数据库按 chapter_id 关联即可。

例如：

```
chapter01_001.mp3
chapter01_002.mp3
```
