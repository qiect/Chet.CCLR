# 使用文档

## 1. 概述

本文档详细描述了Chet.CCLR.WebApi项目中提供的API端点，包括请求方式、参数、响应格式和使用示例。本项目采用RESTful API设计风格，使用JSON作为数据交换格式。

## 2. API基础信息

### 2.1 基础URL

- 开发环境: `http://localhost:5000/api`
- 生产环境: `https://yourdomain.com/api`

### 2.2 认证方式

API使用JWT（JSON Web Token）进行身份验证。请求时需在HTTP头部添加Authorization字段：

```
Authorization: Bearer {token}
```

### 2.3 内容类型

- 请求: `application/json`
- 响应: `application/json`

### 2.4 响应格式

所有API响应都遵循统一的响应模型：

```csharp
public class ApiResponse<T>
{
    public int StatusCode { get; set; }
    public bool Success { get; set; }
    public string? Message { get; set; }
    public T? Data { get; set; }
    public DateTime Timestamp { get; set; }
}
```

## 5. 分页响应格式

当API返回分页数据时，使用以下格式：

```csharp
public class PaginatedResponse<T>
{
    public IEnumerable<T> Items { get; set; } = Array.Empty<T>();
    public int Total { get; set; }
    public int Page { get; set; }
    public int PageSize { get; set; }
    public int TotalPages { get; set; }
    public bool HasPreviousPage => Page > 1;
    public bool HasNextPage => Page < TotalPages;
}
```

## 6. API使用示例

### 6.1 JavaScript/TypeScript客户端示例

```typescript
class ApiClient {
  private baseUrl: string;
  private token: string | null;

  constructor(baseUrl: string) {
    this.baseUrl = baseUrl;
    this.token = localStorage.getItem('token');
  }

  private async request<T>(endpoint: string, options: RequestInit = {}): Promise<T> {
    const url = `${this.baseUrl}${endpoint}`;
    const headers = {
      'Content-Type': 'application/json',
      ...(this.token && { 'Authorization': `Bearer ${this.token}` }),
      ...options.headers
    };

    const response = await fetch(url, { ...options, headers });
    const data = await response.json();

    if (!response.ok) {
      throw new Error(data.message || '请求失败');
    }

    return data;
  }

  async login(email: string, password: string) {
    const response = await this.request('/auth/login', {
      method: 'POST',
      body: JSON.stringify({ email, password })
    });

    this.token = response.data.token;
    localStorage.setItem('token', this.token);
    return response;
  }

  async getUsers(page: number = 1, pageSize: number = 10) {
    return await this.request(`/users?page=${page}&pageSize=${pageSize}`);
  }

  async getUserById(id: number) {
    return await this.request(`/users/${id}`);
  }
}
```

### 6.2 C#客户端示例

```csharp
public class ApiClient
{
    private readonly HttpClient _httpClient;
    private readonly string _baseUrl;

    public ApiClient(HttpClient httpClient, string baseUrl)
    {
        _httpClient = httpClient;
        _baseUrl = baseUrl;
    }

    public async Task<ApiResponse<AuthResponse>> LoginAsync(LoginRequest request)
    {
        var json = JsonSerializer.Serialize(request);
        var content = new StringContent(json, Encoding.UTF8, "application/json");

        var response = await _httpClient.PostAsync($"{_baseUrl}/auth/login", content);
        var responseContent = await response.Content.ReadAsStringAsync();

        return JsonSerializer.Deserialize<ApiResponse<AuthResponse>>(responseContent);
    }

    public async Task<ApiResponse<PaginatedResponse<UserDto>>> GetUsersAsync(
        int page = 1, 
        int pageSize = 10, 
        bool? isActive = null)
    {
        var queryString = $"?page={page}&pageSize={pageSize}";
        if (isActive.HasValue)
        {
            queryString += $"&isActive={isActive.Value}";
        }

        var request = new HttpRequestMessage(HttpMethod.Get, $"{_baseUrl}/users{queryString}");
        request.Headers.Authorization = new AuthenticationHeaderValue("Bearer", GetToken());

        var response = await _httpClient.SendAsync(request);
        var responseContent = await response.Content.ReadAsStringAsync();

        return JsonSerializer.Deserialize<ApiResponse<PaginatedResponse<UserDto>>>(responseContent);
    }

    private string GetToken()
    {
        // 从配置或存储中获取JWT token
        return "your-jwt-token-here";
    }
}

public class LoginRequest
{
    public string Email { get; set; } = string.Empty;
    public string Password { get; set; } = string.Empty;
}

public class AuthResponse
{
    public string Token { get; set; } = string.Empty;
    public UserDto User { get; set; } = new UserDto();
}
```

## 7. API版本管理

### 7.1 版本控制策略

本API采用URL路径版本控制：

- `/api/v1/users` - 第1版API
- `/api/v2/users` - 第2版API

### 7.2 版本声明

在 `Program.cs` 中配置API版本：

```csharp
builder.Services.AddApiVersioning(options =>
{
    options.AssumeDefaultVersionWhenUnspecified = true;
    options.DefaultApiVersion = new ApiVersion(1, 0);
    options.ReportApiVersions = true;
});
```

## 8. API安全注意事项

### 8.1 认证与授权

- 所有敏感操作都需要有效的JWT Token
- 使用HTTPS协议确保传输安全
- 定期刷新JWT Token

### 8.2 输入验证

- 所有输入参数都经过严格验证
- 防止SQL注入和XSS攻击
- 使用参数化查询

### 8.3 速率限制

- 实施API速率限制防止滥用
- 单个IP地址限制请求频率
