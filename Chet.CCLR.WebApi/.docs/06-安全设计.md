# 安全设计

## 1. 安全概述

本文档详细描述了Chet.CCLR.WebApi项目的安全设计，包括身份验证、授权、数据保护和安全最佳实践。本设计采用JWT（JSON Web Token）作为主要的身份验证机制，结合多种安全措施确保API的安全性。

## 2. 身份验证机制

### 2.1 JWT认证流程

```
客户端请求 -> 验证凭据 -> 生成JWT -> 返回令牌 -> 后续请求携带令牌 -> 验证令牌 -> 授权访问
```

### 2.2 JWT配置

在 `appsettings.json` 中配置JWT参数：

```json
{
  "JwtSettings": {
    "Secret": "YourSuperSecretKeyThatShouldBeAtLeast32CharactersLong",
    "Issuer": "Chet.CCLR.WebApi",
    "Audience": "Chet.CCLR.WebApi.Audience",
    "TokenExpirationInMinutes": 60
  }
}
```

### 2.3 JWT实现

```csharp
public class JwtService
{
    private readonly IConfiguration _configuration;

    public JwtService(IConfiguration configuration)
    {
        _configuration = configuration;
    }

    public string GenerateToken(User user)
    {
        var secret = _configuration["JwtSettings:Secret"];
        var issuer = _configuration["JwtSettings:Issuer"];
        var audience = _configuration["JwtSettings:Audience"];
        var expirationInMinutes = int.Parse(_configuration["JwtSettings:TokenExpirationInMinutes"]);

        var key = new SymmetricSecurityKey(Encoding.UTF8.GetBytes(secret));
        var credentials = new SigningCredentials(key, SecurityAlgorithms.HmacSha256);

        var claims = new[]
        {
            new Claim(ClaimTypes.NameIdentifier, user.Id.ToString()),
            new Claim(ClaimTypes.Name, user.Name),
            new Claim(ClaimTypes.Email, user.Email),
            new Claim(JwtRegisteredClaimNames.Jti, Guid.NewGuid().ToString()),
            new Claim(JwtRegisteredClaimNames.Iat, DateTimeOffset.UtcNow.ToUnixTimeSeconds().ToString(), ClaimValueTypes.Integer64)
        };

        var token = new JwtSecurityToken(
            issuer: issuer,
            audience: audience,
            claims: claims,
            expires: DateTime.UtcNow.AddMinutes(expirationInMinutes),
            signingCredentials: credentials);

        return new JwtSecurityTokenHandler().WriteToken(token);
    }

    public ClaimsPrincipal ValidateToken(string token)
    {
        var secret = _configuration["JwtSettings:Secret"];
        var issuer = _configuration["JwtSettings:Issuer"];
        var audience = _configuration["JwtSettings:Audience"];

        var tokenHandler = new JwtSecurityTokenHandler();
        var key = Encoding.UTF8.GetBytes(secret);

        var validationParameters = new TokenValidationParameters
        {
            ClockSkew = TimeSpan.Zero, // 默认: 5 min
            ValidateIssuer = true,
            ValidateAudience = true,
            ValidateLifetime = true,
            ValidateIssuerSigningKey = true,
            ValidIssuer = issuer,
            ValidAudience = audience,
            IssuerSigningKey = new SymmetricSecurityKey(key)
        };

        return tokenHandler.ValidateToken(token, validationParameters, out _);
    }
}
```

### 3.2 策略基础授权

```csharp
// 在Program.cs中配置授权策略
builder.Services.AddAuthorization(options =>
{
    options.AddPolicy("RequireAdministratorRole", policy => policy.RequireRole("Admin"));
    options.AddPolicy("RequireActiveUser", policy => 
        policy.RequireAuthenticatedUser()
              .RequireClaim("IsActive", "true"));
    options.AddPolicy("MinimumAgeRequirement", policy =>
        policy.RequireClaim("Age")
              .RequireAssertion(context =>
              {
                  var ageClaim = context.User.FindFirst(c => c.Type == "Age");
                  if (ageClaim == null) return false;
                  var age = int.Parse(ageClaim.Value);
                  return age >= 18;
              }));
});
```

## 4. 密码安全

### 4.1 密码哈希

使用BCrypt进行密码哈希：

```csharp
public class PasswordService
{
    public static string HashPassword(string password)
    {
        // 生成盐并哈希密码
        return BCrypt.Net.BCrypt.HashPassword(password, BCrypt.Net.BCrypt.GenerateSalt(12));
    }

    public static bool VerifyPassword(string password, string hash)
    {
        // 验证密码
        return BCrypt.Net.BCrypt.Verify(password, hash);
    }
}
```

### 4.2 密码强度验证

```csharp
public static class PasswordValidator
{
    public static bool IsValidPassword(string password)
    {
        // 验证密码至少8个字符，包含大写字母、小写字母、数字和特殊字符
        var hasMinLength = password.Length >= 8;
        var hasUpperCase = Regex.IsMatch(password, @"[A-Z]");
        var hasLowerCase = Regex.IsMatch(password, @"[a-z]");
        var hasNumbers = Regex.IsMatch(password, @"\d");
        var hasSpecialChar = Regex.IsMatch(password, @"[^a-zA-Z0-9]");

        return hasMinLength && hasUpperCase && hasLowerCase && hasNumbers && hasSpecialChar;
    }
}
```

## 5. API安全措施

### 5.1 CORS配置

```csharp
// 在Program.cs中配置CORS
builder.Services.AddCors(options =>
{
    options.AddPolicy("AllowSpecificOrigins", policy =>
    {
        policy.WithOrigins("https://yourdomain.com", "http://localhost:3000") // 替换为实际允许的域名
              .AllowAnyHeader()
              .AllowAnyMethod()
              .AllowCredentials(); // 如果需要发送cookies
    });
});

// 应用CORS中间件
app.UseCors("AllowSpecificOrigins");
```

### 5.2 HTTPS强制

## 7. 安全头设置

### 7.1 安全头中间件

```csharp
// 自定义安全头中间件
public class SecurityHeadersMiddleware
{
    private readonly RequestDelegate _next;

    public SecurityHeadersMiddleware(RequestDelegate next)
    {
        _next = next;
    }

    public async Task InvokeAsync(HttpContext context)
    {
        // 设置安全头
        context.Response.Headers.Add("X-Content-Type-Options", "nosniff");
        context.Response.Headers.Add("X-Frame-Options", "DENY");
        context.Response.Headers.Add("X-XSS-Protection", "1; mode=block");
        context.Response.Headers.Add("Referrer-Policy", "strict-origin-when-cross-origin");
        context.Response.Headers.Add("Permissions-Policy", "geolocation=(), microphone=(), camera=()");
        
        await _next(context);
    }
}

// 在Program.cs中注册中间件
app.UseMiddleware<SecurityHeadersMiddleware>();
```

builder.Services.AddRateLimiter(options =>
{
    options.GlobalLimiter = PartitionedLimiter.Create<HttpContext, string>((_, _) =>
        SlidingWindowPartitionedRateLimiterOptions.Create(
            permitLimit: 100,
            window: TimeSpan.FromMinutes(1),
            segments: 1,
            queueLimit: 0,
            queueProcessingOrder: QueueProcessingOrder.OldestFirst));

    // IP地址限流
    options.AddPartitionedLimiter<HttpContext, string>("IpRateLimiter", context =>
        TypedLimiter.Create(
            new[] { "Ip:" + context.Connection.RemoteIpAddress?.ToString() ?? "unknown" },
            partitionedOptions: SlidingWindowPartitionedRateLimiterOptions.Create(
                permitLimit: 10,
                window: TimeSpan.FromMinutes(1),

                queueProcessingOrder: QueueProcessingOrder.OldestFirst)));
});

## 11. 常见安全漏洞防护

### 11.1 SQL注入防护

- 使用参数化查询
- 使用ORM（如EF Core）的LINQ查询
- 避免动态SQL拼接

### 11.2 XSS防护

- 验证和清理用户输入
- 在输出时进行编码
- 设置适当的CSP头

### 11.3 CSRF防护

- 使用防伪令牌
- 验证请求来源
- 对敏感操作使用额外验证

通过实施以上安全措施，可以有效保护API免受各种安全威胁，确保系统的安全性。
